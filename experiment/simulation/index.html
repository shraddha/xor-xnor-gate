CMOS XOR Gate Schematic

Circuit Description

The CMOS XOR gate consists of:

  • PMOS transistors in the pull-up network
  • NMOS transistors in the pull-down network
  • Two input nodes (A and B) connected to transistor gates
  • Output node (Y) connected to both networks

Output is HIGH when inputs are different (A≠B)

Learning Objectives:

  • Understand the relationship between transistor sizes and switching characteristics
  • Learn how to optimize XOR/XNOR gate performance through parameter adjustment
  • Analyze the impact of supply voltage on circuit behavior
  • Learn SPICE syntax and circuit description through hands-on editing
  • Develop skills in writing and validating SPICE netlists
  • Understand common SPICE errors and how to fix them

XOR Gate Parameters

1V 5V 5V
Hint: Higher VDD increases power consumption but improves noise margins
0.5μm 2.0μm 10.0μm
Hint: PMOS width affects rise time and high-to-low transition
0.5μm 1.0μm 10.0μm
Hint: NMOS width affects fall time and low-to-high transition
0.1μm 0.18μm 1.0μm
Hint: Channel length affects current drive and switching speed

Note on W/L Ratio

The typical Wp/Wn ratio for a symmetrical CMOS XOR/XNOR gate is approximately 2:1 to compensate for the lower hole mobility in PMOS compared to electron mobility in NMOS.
